Expand E-learning Framework

Expand is an e-learning framework which allows you to build SCORM conformant e-learning courses using web technologies rather than authoring tools.

Benefits of Expand include:

  • Fully responsive
  • Bootstrap included
  • Doesn't reload browser upon every page load
  • SCORM 1.2 conformant
  • Bookmarking automatically handled
  • Easily publish a SCORM zip with a single command line
  • Endless possibilities for development
